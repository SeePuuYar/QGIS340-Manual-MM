31. ပါဝင်အားဖြည့်သူများ (Contributors)

  • ရေးသားပြုစုသူများ (Authors)

  • ဘာသာပြန်ဆိုသူများ (Translators)

QGIS သည် စေတနာ့ဝန်ထမ်းများနှင့် အဖွဲ့အစည်းများမှ ဖန်တီးထားသော open source (အခကြေးငွေ ပေးစရာမလိုပဲ လွတ်လပ်စွာအသုံးပြုနိုင်သော) project တစ်ခုဖြစ်ပါသည်။ QGIS မှ လူမျိုး၊ ဘာသာ၊ ကျား/မနှင့် ဆင်းရဲ/ချမ်းသာ ခွဲခြားခြင်းမရှိဘဲ အားလုံးပါဝင်ဆောင်ရွက်နိုင်သည့်အသိုက်အဝန်းတစ်ခုအဖြစ် ကြိုဆိုလျက်ရှိပါသည်။ မည်သည့်အခြေအနေတွင်မဆို get involved သို့ဝင်ရောက်၍ ပါဝင်ဆောင်ရွက်နိုင်ပါသည်။

31.1. ရေးသားပြုစုသူများ (Authors)

31.2. ဘာသာပြန်ဆိုသူများ (Translators)

QGIS သည် ဘာသာစကားပေါင်းစုံနှင့်အသုံးပြုနိုင်သော application တစ်ခုဖြစ်ပြီး သုံးစွဲနည်းလမ်းညွှန်ကိုလည်း ဘာသာစကားပေါင်းများစွာဖြင့် ထုတ်ဝေလျက်ရှိပါသည်။ ကျန် အခြားဘာသာစကားများစွာသို့ လည်း ပြန်ဆိုလျက်ရှိပြီး သင့်တော်သည့် ဘာသာပြန်ခြင်းလုပ်ငန်းပမာဏပြီးစီးသည့်အခါ ထုတ်ဝေမည်ဖြစ်ပါသည်။ QGIS တွင်အသုံးပြုနေသော ဘာသာစကားတစ်ခု တိုးတက်စေရန် ကူညီလိုပါက သို့မဟုတ် အသစ်တစ်ခု တောင်းဆိုလိုပါက Translation Guidelines သို့ဝင်ရောက်ကြည့်ရှုနိုင်ပါသည်။

လက်ရှိဘာသာပေါင်းစုံသို့ ပြန်ဆိုမှုလုပ်ငန်းများအတွက်ကျေးဇူးတင်ထိုက်သူများမှာ-
