27.10. Processing Algorithm များအတွက် Input နှင့် Output အမျိုးအစားများ (Input and output types for Processing Algorithms)

သက်ဆိုင်သော alg decorator ကိန်းသေများပါဝင်ပြီး processing ထဲတွင် လုပ်ဆောင်နိုင်သော Input နှင့် Output အမျိုးအစားများစာရင်းကို အောက်တွင် ဖော်ပြထားပါသည်။

27.10.1. Input types (ထည့်သွင်းအသုံးပြုသည့်အချက်အလက် အမျိုးအစားများ)

Input နာမည်

Class

Alg constant

ရှင်းလင်းဖော်ပြချက်

Annotation (မှတ်စာ) layer

QgsProcessingParameterAnnotationLayer

alg.ANNOTATION_LAYER

Annotation (မှတ်စာ) layer တစ်ခု

Area (ဧရိယာ)

QgsProcessingParameterArea

ဧရိယာအတိုင်းအတာတစ်ခုကို ကိုယ်စားပြုသော ကိန်းဂဏန်း parameter တစ်ခု

Authentication Configuration (အစစ်အမှန်ဖြစ်ခြင်းဆိုင်ရာ ပြင်ဆင်သတ်မှတ်ခြင်း)

QgsProcessingParameterAuthConfig

alg.AUTH_CFG

ပေးထားသော authentication configuration များမှ ရွေးချယ်နိုင်ပါသည် သို့မဟုတ် authentication configuration အသစ်ကို ဖန်တီးနိုင်ပါသည်။
