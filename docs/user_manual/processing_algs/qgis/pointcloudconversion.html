  • Format ပြောင်းလဲခြင်း (Convert format)

  • Raster အဖြစ် ထုတ်ယူခြင်း (Export to raster)

  • Raster အဖြစ်သို့ ထုတ်ယူခြင်း (Triangulation ကိုအသုံးပြု၍) (Export raster (using triangulation))

  • Vector အဖြစ်သို့ ထုတ်ယူခြင်း (Export to vector)

Note

QGIS သည် PDAL library version 2.5.0 သို့မဟုတ် version အသစ်ကို အသုံးပြုမှသာ ဤ algorithm များကို ရရှိနိုင်မည်ဖြစ်သည်။

28.1.12.1. Format ပြောင်းလဲခြင်း (Convert format)

Point cloud တစ်ခုအား မတူညီသော ဖိုင် format တစ်ခုသို့ပြောင်းလဲပေးပါသည်၊ ဥပမာ- compress (ချုံ့ထားသော) လုပ်ထားသော .LAZ တစ်ခုကို ဖန်တီးပေးပါသည်။

သတ်မှတ်ချက်များ (Parameters)

အညွှန်း

အမည်

အမျိုးအစား

ရှင်းလင်းဖော်ပြချက်

Input layer (ထည့်သွင်းအသုံးပြုသော layer)

INPUT

[point cloud]

ပြောင်းလဲမည့် input point cloud layer

Converted (ပြောင်းလဲထားပြီးသော)

OUTPUT

[point cloud]

Default: [Save to temporary file]

Output အနေဖြင့်အသုံးပြုမည့် point cloud ဖိုင်ကို သတ်မှတ်ပါ။ အောက်ပါတို့ထဲမှ တစ်ခုခုဖြစ်ပါသည်-

  • ယာယီ file တစ်ခုတွင် သိမ်းဆည်းပါ

  • File တွင်သိမ်းဆည်းပါ

ရလာဒ်များ (Outputs)

အညွှန်း

အမည်

အမျိုးအစား

ရှင်းလင်းဖော်ပြချက်

Converted (ပြောင်းလဲထားပြီးသော)

OUTPUT

[point cloud]

မွမ်းမံပြင်ဆင်ထားသော ဖိုင် format တစ်ခုဖြင့် output point cloud layer။ လက်ရှိ ရရှိနိုင်သော format များမှာ .LAS ၊ .LAZ ၊ .COPC.LAZ နှင့် .VPC တို့ဖြစ်ပါသည်။

Python code

Algorithm ID: pdal:convertformat

import processing
processing.run("algorithm_id", {parameter_dictionary})

Processing Toolbox ထဲရှိ algorithm ပေါ်တွင် mouse ကိုတင်ထားလျှင် algorithm id ကိုပြသပေးပါသည်။ parameter dictionary သည် parameter နာမည်များနှင့် တန်ဖိုးများကို ပေးပါသည်။ Python console မှ algorithm များကို မည်သို့လုပ်ဆောင်ရမည် ဆိုသည့် အသေးစိတ်ကို သိရှိလိုလျှင် Console တွင် processing algorithm များကို အသုံးပြုခြင်း (Using processing algorithms from the console) တွင်ကြည့်ပါ။

28.1.12.2. Raster အဖြစ် ထုတ်ယူခြင်း (Export to raster)

Point cloud data အား 2D raster grid တစ်ခုအဖြစ်သို့ ထုတ်ယူပေးပါသည်၊ raster grid တွင် ပေးထားသော resolution အတိုင်း cell အရွယ်အစားရှိမည်ဖြစ်ပြီး သတ်မှတ်ပေးထားသော attribute မှ တန်ဖိုးများကို ထည့်သွင်းရေးသားပေးပါသည်။

သတ်မှတ်ချက်များ (Parameters)

အခြေခံသတ်မှတ်ချက်များ (Basic parameters)

အညွှန်း

အမည်

အမျိုးအစား

ရှင်းလင်းဖော်ပြချက်

Input layer (ထည့်သွင်းအသုံးပြုသော layer)

INPUT

[point cloud]

Export ထုတ်ယူမည့် input point cloud layer

Attribute (အချက်အလက်များ)

ATTRIBUTE

[field] [enumeration]

တန်ဖိုးများထုတ်ယူမည့် point cloud layer ၏ field တစ်ခု

Resolution of the density raster (Density raster ၏ ကြည်လင်ပြတ်သားမှု)

RESOLUTION

[number]

Default: 1.0

Output raster ၏ cell အရွယ်အစား

Tile size for parallel runs (အပြိုင် run ခြင်းများအတွက် Tile အရွယ်အစား)

TILE_SIZE

[number]

Default: 1000

Exported (ထုတ်ယူထားပြီးသော)

OUTPUT

[raster]

Default: [Save to temporary file]

Data များကိုထုတ်ယူမည့် raster ဖိုင်ကို သတ်မှတ်ပါ။ အောက်ပါတို့ထဲမှ တစ်ခုခုဖြစ်ပါသည်-

  • ယာယီ file တစ်ခုတွင် သိမ်းဆည်းပါ

  • File တွင်သိမ်းဆည်းပါ

အဆင့်မြင့်သတ်မှတ်ချက်များ (Advanced parameters)

အညွှန်း

အမည်

အမျိုးအစား

ရှင်းလင်းဖော်ပြချက်

Filter expression

Optional (မဖြစ်မနေလုပ်ဆောင်ရန်မလိုပါ)

FILTER_EXPRESSION

[expression]

Point cloud data ထဲရှိ feature များအစုတစ်ခုကို ရွေးချယ်ရန်အတွက် PDAL expression တစ်ခု။

Cropping extent (ဖြတ်ထုတ်မည့် နယ်ပယ်အကျယ်အဝန်း)

Optional (မဖြစ်မနေလုပ်ဆောင်ရန်မလိုပါ)

FILTER_EXTENT

[extent]

Point cloud data ထဲရှိ feature များအစုတစ်ခုကို ရွေးချယ်ရန်အတွက် မြေပုံ နယ်ပယ်အကျယ်အဝန်း (extent)။

အသုံးပြုနိုင်သော နည်းလမ်းများမှာ -

  • Layer မှ တွက်ချက်ခြင်း - လက်ရှိ project ထဲတွင်ထည့်သွင်းထားသော layer ၏ extent (နယ်ပယ်အကျယ်အဝန်း)ကိုအသုံးပြုပါသည်။

  • Layout map မှ တွက်ချက်ခြင်း - အသုံးပြုနေသော project ထဲရှိ layout map item တစ်ခု၏ extent အသုံးပြုပါသည်။

  • Bookmark မှ တွက်ချက်ခြင်း - မှတ်ထားသော bookmark တစ်ခု၏ extent ကိုအသုံးပြုပါသည်။

  • မြေပုံ canvas extent ကိုအသုံးပြုခြင်း

  • Canvas ပေါ်တွင်ရေးဆွဲခြင်း - တွက်ချက်လိုသည့် ဧရိယာကို ထောင့်မှန်စတုဂံပုံဆွဲခြင်း။

  • x အနည်းဆုံး၊ x အများဆုံး၊ y အနည်းဆုံး၊  y အများဆုံး တို့အဖြစ် ကိုဩဒိနိတ်များကို ထည့်သွင်းခြင်း။

X origin of a tile for parallel runs (အပြိုင် run ခြင်းများအတွက် tile တစ်ခု၏ X မူလအစ)

Optional (မဖြစ်မနေလုပ်ဆောင်ရန်မလိုပါ)

ORIGIN_X

[number]

Y origin of a tile for parallel runs (အပြိုင် run ခြင်းများအတွက် tile တစ်ခု၏ Y မူလအစ)

Optional (မဖြစ်မနေလုပ်ဆောင်ရန်မလိုပါ)

ORIGIN_Y

[number]

ရလာဒ်များ (Outputs)

အညွှန်း

အမည်

အမျိုးအစား

ရှင်းလင်းဖော်ပြချက်

Exported

OUTPUT

[raster]

Point cloud layer ၏ feature များကို ထုတ်ယူထားသော output raster layer ။ လက်ရှိ ရရှိနိုင်သော format မှာ .TIF ဖြစ်ပါသည်။

Python code

import processing

Processing Toolbox ထဲရှိ algorithm ပေါ်တွင် mouse ကိုတင်ထားလျှင် algorithm id ကိုပြသပေးပါသည်။ parameter dictionary သည် parameter နာမည်များနှင့် တန်ဖိုးများကို ပေးပါသည်။ Python console မှ algorithm များကို မည်သို့လုပ်ဆောင်ရမည် ဆိုသည့် အသေးစိတ်ကို သိရှိလိုလျှင် Console တွင် processing algorithm များကို အသုံးပြုခြင်း (Using processing algorithms from the console) တွင်ကြည့်ပါ။

28.1.12.3. Raster အဖြစ်သို့ ထုတ်ယူခြင်း (Triangulation ကိုအသုံးပြု၍) (Export raster (using triangulation))

Point များကို (triangulation) တြိဂံဖွဲ့ပြီးနောက် တြိဂံများမှ cell တန်ဖိုးများကို interpolate (သွယ်ဝိုက်တွက်ချက်) လုပ်ပြီး point cloud data များကို 2D raster grid တစ်ခုအဖြစ်သို့ export ထုတ်ပေးပါသည်။

Note

ကြီးမားသော Dataset များအတွက် ဤ algorithm ကိုအသုံးပြုပါက နှေးကွေးစေနိုင်ပါသည်။ Point cloud သည် သိပ်သည်းမှုများပါက Raster အဖြစ် ထုတ်ယူခြင်း (Export to raster) algorithm အသုံးပြုပြီး မြေပြင် point များကို raster တစ်ခုအနေဖြင့် export ထုတ်ယူနိုင်ပါသည်။
