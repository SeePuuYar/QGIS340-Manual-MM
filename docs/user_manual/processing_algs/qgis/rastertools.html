28.1.18.1. Raster တန်းညှိခြင်း (Align raster)

Reference raster တစ်ခုအတိုင်း တူညီသော cell အရွယ်အစားအဖြစ်သို့ resolution ပြောင်းလဲ (resample) ပေးပြီး တူညီသော CRS သို့ပြောင်းလဲပေးခြင်းဖြင့် Raster ကို align လုပ်ပေးပါသည်။

Warning

ဤ algorithm သည် Model Designer တွင်သာအသုံးပြုနိုင်ပါသည်။ အခြားအရာများအတွက် Raster များ တန်းညှိခြင်း (Align rasters) ကိုအသုံးပြုပါ။

သတ်မှတ်ချက်များ (Parameters)

အညွှန်း

အမည်

အမျိုးအစား

ရှင်းလင်းဖော်ပြချက်

Input layer (ထည့်သွင်းအသုံးပြုသော layer)

INPUT

[raster]

Align ပြုလုပ်မည့် input raster layer

Resampling method (Resampling နည်းလမ်း)

RESAMPLING_METHOD

[enumeration]

Default: 0

Input layer ကို resampling လုပ်ခြင်းအတွက် အသုံးပြုမည့် နည်းလမ်း ရရှိနိုင်သောရွေးချယ်စရာများ မှာ-

  • 0 — Nearest Neighbour

  • 1 — Bilinear (2x2 kernel)

  • 2 — Cubic (4x4 kernel)

  • 3 — Cubic B-Spline (4x4 kernel)

  • 4 — Lanczos (6x6 kernel)

  • 5 — Average

  • 6 — Mode

  • 7 — Maximum

  • 8 — Minimum

  • 9 — Median

  • 10 — First quartile (Q1)

  • 11 — Third quartile (Q3)

Rescale values according to the cell size (Cell အရွယ်အစားအရ rescale တန်ဖိုးများ)

RESCALE

[boolean]

Default: No

Reference layer

REFERENCE_LAYER

[raster]

Input layer များတွင် အသုံးချရန် extent ၊ cell အရွယ်အစားနှင့် CRS များကိုရယူရာတွင် အသုံးပြုမည့် raster layer

Override reference CRS

Optional (မဖြစ်မနေလုပ်ဆောင်ရန် မလိုပါ)

CRS

[crs]

Reference layer ၏ CRS အစား အသုံးပြုမည့် CRS

Override reference cell size X

Optional (မဖြစ်မနေလုပ်ဆောင်ရန် မလိုပါ)

CELL_SIZE_X

[number]

Reference layer ၏ cell size အစား အသုံးပြုမည့် X လားရာရှိ cell အရွယ်အစား

Override reference cell size Y

Optional (မဖြစ်မနေလုပ်ဆောင်ရန် မလိုပါ)

CELL_SIZE_Y

[number]

Reference layer ၏ cell size အစား အသုံးပြုမည့် Y လားရာရှိ cell အရွယ်အစား

Override reference grid offset X

Optional (မဖြစ်မနေလုပ်ဆောင်ရန် မလိုပါ)

GRID_OFFSET_X

[number]

Cells grid တွင် အသုံးပြုမည့် X လားရာရှိ Offset

Override reference grid offset Y

Optional (မဖြစ်မနေလုပ်ဆောင်ရန် မလိုပါ)

GRID_OFFSET_Y

[number]

Cells grid တွင် အသုံးပြုမည့် Y လားရာရှိ Offset

Clip to extent

Optional (မဖြစ်မနေလုပ်ဆောင်ရန် မလိုပါ)

EXTENT

[extent]

Output raster layer ၏ extent ကိုသတ်မှတ်ပါ။ ၎င်းကို အတွင်းပိုင်းတွင် tile အရွယ်အစားအများအပြားသို့ ချဲ့ထွင်ပေးပါလိမ့်မည်။

အသုံးပြုနိုင်သော နည်းလမ်းများမှာ -

  • Layer မှ တွက်ချက်ခြင်း - လက်ရှိ project ထဲတွင်ထည့်သွင်းထားသော layer ၏ extent (နယ်ပယ်အကျယ်အဝန်း)ကိုအသုံးပြုပါသည်။

  • Layout map မှ တွက်ချက်ခြင်း - အသုံးပြုနေသော project ထဲရှိ layout map item တစ်ခု၏ extent အသုံးပြုပါသည်။

  • Bookmark မှ တွက်ချက်ခြင်း - မှတ်ထားသော bookmark တစ်ခု၏ extent ကိုအသုံးပြုပါသည်။

  • မြေပုံ canvas extent ကိုအသုံးပြုခြင်း

  • Canvas ပေါ်တွင်ရေးဆွဲခြင်း - တွက်ချက်လိုသည့် ဧရိယာကို ထောင့်မှန်စတုဂံပုံဆွဲခြင်း။

  • x အနည်းဆုံး၊ x အများဆုံး၊ y အနည်းဆုံး၊  y အများဆုံး တို့အဖြစ် ကိုဩဒိနိတ်များကို ထည့်သွင်းခြင်း။

Aligned raster

OUTPUT

import processing
processing.run("algorithm_id", {parameter_dictionary})

Processing Toolbox ထဲရှိ algorithm ပေါ်တွင် mouse ကိုတင်ထားလျှင် algorithm id ကိုပြသပေးပါသည်။ parameter dictionary သည် parameter နာမည်များနှင့် တန်ဖိုးများကို ပေးပါသည်။ Python console မှ algorithm များကို မည်သို့လုပ်ဆောင်ရမည် ဆိုသည့် အသေးစိတ်ကို သိရှိလိုလျှင် Console တွင် processing algorithm များကို အသုံးပြုခြင်း (Using processing algorithms from the console) တွင်ကြည့်ပါ။

28.1.18.3. မြေပုံအား raster သို့ ပြောင်းလဲခြင်း (Convert map to raster)

Map canvas content (မြေပုံ canvas အကြောင်းအရာ) ၏ raster ဓာတ်ပုံတစ်ခုကို ဖန်တီးပေးပါသည်။

Layer တစ်ခုချင်းစီအတွက် သတ်မှတ်ထားသော style တစ်ခုနှင့်အတူ ကြိုတင်သတ်မှတ်ထားသည့် layer များအစု တစ်ခုကို render (ပုံဖော်ပြသခြင်း) ပြုလုပ်ရန် map theme (မြေပုံခေါင်းစဉ်) တစ်ခုကို ရွေးချယ်နိုင်ပါသည်။

တနည်းအားဖြင့်၊ map theme (မြေပုံခေါင်းစဉ်) ကို သတ်မှတ်မထားပါက single layer တစ်ခုကို ရွေးချယ်နိုင်ပါသည်။

အကယ်၍ map theme (မြေပုံခေါင်းစဉ်) သို့မဟုတ် layer တစ်ခုကိုမျှသတ်မှတ်ထားခြင်းမရှိပါက လက်ရှိ မြေပုံ content (အကြောင်းအရာ) ကိုသာ ပုံဖော်ပြသခြင်း (render) ပြုလုပ်မည်ဖြစ်ပါသည်။ ထည့်သွင်းထားသည့် အနိမ့်ဆုံး အကျယ်အဝန်းနယ် (minimum extent) ကို tile အရွယ်အစား အများအပြားအဖြစ် အတွင်းပိုင်း (internal) တွင် တိုးချဲ့ (extend) ပေးသွားမည်ဖြစ်ပါသည်။

28.1.18.5. XYZ tile များဖန်တီးခြင်း (Directory ပုံစံ)

လက်ရှိ QGIS project ကို အသုံးပြုပြီး raster “XYZ” tile များကို individual (သီးခြား) image များအဖြစ် directory structure (hierarchy အရ ရှိသည့် folder များထဲရှိ ဖိုင်များ၏ ဖွဲ့စည်းတည်ဆောက်ပုံ) တစ်ခုထဲသို့ ထုတ်လုပ်ပေးပါသည်။

စိတ်ကြိုက်ရွေးချယ်မှုအနေဖြင့် ထုတ်လုပ်ထားသည့် tile များ (generated tiles) ကို အသုံးပြုပြီး Leaflet (web mapping application များတည်ဆောက်ရန်အသုံးပြုသည့် open source JavaScript library) HTML output file တစ်ခုကို မြေပုံ layer တစ်ခုအဖြစ် ဖန်တီးနိုင်ပါသည်။
