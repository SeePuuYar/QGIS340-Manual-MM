21.1. 3D Tile ဆိုသည်မှာ ဘယ်လိုအရာတွေလဲ (What are 3D Tiles?)

3D tile များသည် အကြီးစား 3D ပထဝီဝင်ဆိုင်ရာ dataset များကို တိုက်ရိုက်ကြည့်ရှုခြင်းနှင့် ပုံဖော်ပြသခြင်းများအတွက် သတ်မှတ်ချက်များဖြစ်ပါသည်။ ၎င်းတို့သည် 3D အကြောင်းအရာများကို ထိထိရောက်ရောက် စီမံနိုင်ရန်နှင့် ပြသနိုင်ရန်အတွက် hierarchical structure (အစဉ်လိုက်စီထားသော ဖွဲ့စည်းပုံ) တစ်ခုကိုအသုံးပြုပြီး အသေးစိတ် level များကို ဆီလျော်အောင်ပြောင်းလဲပြသပေးသောကြောင့် စွမ်းဆောင်ရည်ကို ကောင်းမွန်စေပါသည်။ ဤနည်းပညာကို မြို့ပြစီမံကိန်း၊ ဗိသုကာပညာ၊ သရုပ်သကန်ဖန်တီးခြင်း (simulation) ၊ ဂိမ်းများ နှင့် virtual reality (VR) များတွင် တွင်ကျယ်စွာ အသုံးပြုပြီး ရှုပ်ထွေးသော ပထဝီဝင်ဆိုင်ရာ data များကို ပုံဖော်ကြည့်ရှုရန်အတွက် စံဖြစ်ပြီး အပြန်အလှန်လုပ်ဆောင်နိုင်သော ဖြေရှင်းနည်းတစ်ခုဖြစ်ပါသည်။

QGIS တွင် လက်ရှိ 3D tile format ၂ မျိုးအသုံးပြုနိုင်ပါသည်-

  • Cesium 3D tiles များကို အဆောက်အဦများ သို့မဟုတ် မြို့တစ်မြို့လုံး၏ ရှုပ်ထွေးသော 3D model များအတွက် အဓိက အသုံးပြုပါသည်။ အဆိုပါ dataset များကို Cesium Ion သို့မဟုတ် Google (Photorealistic 3D Tiles) ကဲ့သို့ cloud-based platform များမှ ပံ့ပိုးပေးထားနိုင်ပါသည်။

  • Quantized Mesh tiles များကို မြေမျက်နှာသွင်ပြင် အနိမ့်အမြင့် data အတွက် အသုံးပြုပါသည်။

ဤ data ရင်းမြစ်များကို QGIS ထဲသို့ ထည့်သွင်းနည်းကို 3D tile scene ဝန်ဆောင်မှုများကို အသုံးပြုခြင်း (Using 3D tiled scene services) တွင်ကြည့်ရှုပါ။

21.2.1. သတင်းအချက်အလက်ဂုဏ်သတ္တိများ (Information Properties)

Information tab သည် ဖတ်ရှုရုံသာ ဖြစ်ပြီး လက်ရှိအလွှာ၏ အကျဉ်းချုပ်အချက်အလက်နှင့် Metadata ကို ခပ်မြန်မြန်ကြည့်ရှုနိုင်သည့် စိတ်ဝင်စားစရာနေရာတစ်ခုဖြစ်ပါသည်။ ကြည့်ရှုနိုင်သည့် အချက်အလက်များမှာ-

  • အလွှာဖန်တီးသူပေါ်မူတည်ပြီး - အမည်၊ URI၊ အရင်းအမြစ်ပုံစံနှင့် ဖိုင်လမ်းကြောင်း၊ Zoom-levels အရေအတွက်

  • စိတ်ကြိုက်ဂုဏ်သတ္တိများ- active project ထဲတွင် layer အကြောင်း အချက်အလက်အပိုများ သိမ်းဆည်းလေ့ရှိပါသည်။ နောက်ထပ်ဂုဏ်သတ္တိများကို PyQGIS ၊ အထူးသဖြင့် setCustomProperty() နည်းလမ်းများမှ သီးခြားစီ ဖန်တီးနိုင်ပြီး စီမံခန့်ခွဲနိုင်ပါသည်။

  • Coordinate Reference System - အမည်၊ ယူနစ်များ၊ နည်းလမ်း၊ တိကျမှု၊ အကိုးအကား (ပုံသေအငြိမ်ဖြစ်သော (static) သို့မဟုတ် ပြောင်းလဲနိုင်သော (dynamic) ကိုဆိုလိုပါသည်)

  • ဖြည့်သွင်းထားသည့် metadata မှ ထုတ်ယူထားသော - အသုံးပြုခွင့်၊ အကျယ်အဝန်း၊ အချိတ်အဆက်များ (links)၊ အဆက်အသွယ်၊ data ၏သမိုင်းကြောင်း၊ အစရှိသည်….

21.2.2. ရင်းမြစ်ဆိုင်ရာ ဂုဏ်သတ္တိများ (Source Properties)

Source tab တွင် ရွေးချယ်ထားသော 3D tile နှင့်ပတ်သက်သော အခြေခံသတင်းအချက်အလက်များကို ပြသပေးပါသည်၊ အောက်ပါတို့ပါဝင်ပါသည်-

  • Layers Panel ထဲတွင် ပြသမည့် Layer name ၊

  • Coordinate Reference System - layer ၏ Coordinate Reference System (CRS) ကိုပြသပေးပါသည်။ Drop-down စာရင်းထဲတွင် လတ်တလော အသုံးပြုခဲ့သော CRS ကို ရွေးချယ်နိုင်ခြင်းဖြင့် သို့မဟုတ် Select CRS ခလုတ်ကို နှိပ်ခြင်းဖြင့် layer ၏ CRS ကိုပြောင်းလဲပေးနိုင်ပါသည် (ကိုသြဒိနိတ်ရည်ညွှန်းစနစ်ရွေးချယ်ရာ (Coordinate Reference System Selector) တွင်ကြည့်ပါ)။ Layer CRS မှားယွင်းနေမှသာ သို့မဟုတ် CRS သတ်မှတ်ထားခြင်းမရှိမှသာ ဤလုပ်ငန်းစဉ်ကို အသုံးပြုပါ။

21.2.3. သင်္ကေတ ဂုဏ်သတ္တိများ (Symbology Properties)

Default အားဖြင့် texture အသုံးပြုပြီး layer ကိုစတိုင်ပြုလုပ်ပါသည်၊ သို့သော် scene ၏နောက်ကွယ်ရှိ wirefram mesh ကိုကြည့်ရှုနိုင်ရန် drop-down menu ထဲရှိ Wireframe ကိုရွေးချယ်ပြီး ၎င်းစတိုင်ကို ပြောင်းလဲပေးနိုင်ပါသည်။ ထို့အပြင် vector polygon များနှင့်ဆင်တူသော mesh fill နှင့် line သင်္ကေတများကိုလည်း ပြောင်းလဲပေးနိုင်ပါသည်။ Use texture colors ကို အမှန်ခြစ်ပါက mesh element တစ်ခုချင်းကို texture အပြည့်အစုံ၏ ပျမ်းမျှတန်ဖိုးဖြင့် ပုံဖော်ပြသပေးပါလိမ့်မည်။ ဤနည်းလမ်းသည် ကြီးမားသော dataset များနှင့်လုပ်ဆောင်ပြီး data ကို မြန်မြန်ခြုံငုံကြည့်ရှုလိုပါက ကောင်းမွန်သောရွေးချယ်စရာတစ်ခုဖြစ်ပါသည်။

Data ကိုကြည့်ရှုရန် New 3D map view ကိုဖွင့်နိုင်ပါသည်။

Symbology tab တွင် layer ၏ feature များအားလုံးပေါ်တွင် အမြဲသက်ရောက်မည့် ရွေးချယ်စရာအချို့ကိုလည်း သတ်မှတ်ပေးနိုင်ပါသည်-

  • Maximum error (အများဆုံးအမှား) - ဤ parameter သည် 3D model ထဲတွင် ပြသသော အသေးစိတ် level ကို ဆုံးဖြတ်ပေးပါသည်။ Point cloud နှင့်အလားတူပင် 3D tile များတွင် ပုံဖော်ပြသခြင်းအတွက် လိုအပ်သည်ထက်ပိုသော အချက်အလက်များ ပိုမိုပါဝင်လေ့ရှိပါသည်။ ဤ setting ကို ချိန်ညှိခြင်းဖြင့် ပြသခြင်း သိပ်သည်းမှု (display density) နှင့် ပုံဖော်ပြသမှုနှုန်း (rendering speed) အကြား မျှခြေကို ထိန်းချုပ်နိုင်ပါသည်။ တန်ဖိုးအကြီးတစ်ခု (ဥပမာ- 5 mm) သည် element များအကြား သိသာသော gap များရှိစေနိုင်ပြီး တန်ဖိုးအသေးတစ်ခု (ဥပမာ- 0.1 mm) သည် အသေးစိတ်အချက်အလက်အများအပြားကို ပုံဖော်ပြသပေးကာ rendering process ကို နှေးကွေးသွားစေနိုင်ပါသည်။ သင်၏ သီးသန့်လိုအပ်ချက်များအတိုင်း setting တွင် မတူညီသော ယူနစ်များကို ရွေးချယ်ပေးနိုင်ပါသည်။

  • Opacity (အလင်းပိတ်နှုန်း) - Map canvas ထဲရှိ အောက်တွင်ရှိနေသော layer ကို မြင်ရစေရန် ဒီ tool ကို အသုံးပြုနိုင်ပါသည်။ မိမိကြည့်လိုသော မြင်ရနိုင်မှု အတိုင်းအတာအထိ slider ကို ရွှေ့ကြည့်ပါ။ မြင်ရနိုင်စွမ်း ရာခိုင်နှုန်းအတိအကျ သတ်မှတ်ပေးလိုလျှင် slider ၏ဘေးရှိ menu တွင် သတ်မှတ်ပေးနိုင်ပါသည်။

  • Blending mode (ရောစပ်ခြင်း) - ဤ tool ကိုအသုံးပြုပြီး အထူး rendering effect များကိုလုပ်ဆောင်နိုင်ပါသည်။ အထက်နှင့်အောက် ထပ်နေသော layer များ၏ pixel များကို ရောစပ်ခြင်းနည်းလမ်းများ (Blending Modes) တွင်ဖော်ပြထားသော setting များဖြင့် ရောစပ်လိုက်ပါသည်။

21.2.4. 3D မြင်ကွင်းဂုဏ်သတ္တိများ (3D View Properties)

  • Maximum screen space error သည် terrain tile များကို ပိုမိုအသေးစိတ်သောအရာများဖြင့် နေရာလဲလှယ်ရန်အတွက် threshold (သတ်မှတ်ချက်) ကိုဆုံးဖြတ်ပေးပါသည် (အပြန်အလှန်အားဖြင့် ပိုမိုအသေးစိတ်သောအရာများကို terrain tile များဖြင့် နေရာလဲလှယ်ခြင်း) - ဆိုလိုသည်မှာ 3D မြင်ကွင်းသည် အရည်အသွေးမြင့် tile များကို မည်မျှကြာကြာ အသုံးပြုမည်ကို ဆုံးဖြတ်ပေးပါသည်။ နံပါတ် အနိမ့်သည် မြင်ကွင်းတွင် ပိုမိုအသေးစိတ်ပြသပြီး rendering ကိုပိုမိုရှုပ်ထွေးလာစေပါသည်။

  • Show bounding boxes သည် terrain tile များ၏ 3D လေးမျက်နှာအကျယ်အဝန်းနယ်များ (bounding box) ကို ပြသပေးပါသည် (terrain ပြဿနာများကို ဖြေရှင်းခြင်းအတွက် အသုံးဝင်ပါသည်)။

21.2.5. ပုံဖော်ပြသခြင်းဆိုင်ရာ ဂုဏ်သတ္တိများ (Rendering Properties)

Scale dependent visibility (စကေးပေါ်မူတည်သည့် မြင်ရနိုင်မှု) အောက်တွင် အများဆုံး (သူ့အောက်ဆိုရင် မြင်ရ) နှင့် အနည်းဆုံး (သူ့အောက်ဆိုရင် မမြင်ရ) စကေးကို သတ်မှတ်ပေးနိုင်ပါသည်။ ၎င်းသည် mesh element များကိုမြင်ရနိုင်သော စကေးအပိုင်းအခြားကို သတ်မှတ်ပေးမည်ဖြစ်သည်။ ထိုစကေးအပိုင်းအခြားအတွင်း မဝင်လျှင် layer များကို မြင်ရမည်မဟုတ်ပါ။ Set to current canvas scale (လက်ရှိမြေပုံ canvas စကေးအတိုင်းသတ်မှတ်ခြင်း) ခလုတ်ကိုနှိပ်ပြီး လက်ရှိမြေပုံ canvas စကေးကို မြင်ရနိုင်သည့် စကေးအပိုင်းအခြားအဖြစ်သတ်မှတ်ပေးနိုင်ပါသည်။ ပိုမိုသိရှိလိုသည်များအတွက် မြင်ရနိုင်သောစကေး ရွေးချယ်ပေးသည့်အရာ (Visibility Scale Selector) တွင် ကြည့်ရှုပါ။

21.2.6. မြေပြင်အနိမ့်အမြင့်ဆိုင်ရာ ဂုဏ်သတ္တိများ (Elevation Properties)

Elevation tab တွင် 3D map view တစ်ခုအတွင်း layer elevation ဂုဏ်သတ္တိများကို ထိန်းချုပ်နိုင်ရန် ရွေးချယ်စရာများပါဝင်ပါသည်။ အထူးသဖြင့် အောက်ပါကို သတ်မှတ်ပေးနိုင်ပါသည်-

  • Elevation Surface - 3D layer vertex များ၏ Z တန်ဖိုးများကို terrain elevation (မြေမျက်နှာသွင်ပြင် အနိမ့်အမြင့်) အဖြစ် မည်ကဲ့သို့ အဓိပ္ပါယ်ဖော်သင့်သည်ကို ဆုံးဖြတ်ပေးပါသည်။ Scale factor တစ်ခု နှင့် Offset တစ်ခုကို အသုံးပြုပေးနိုင်ပါသည်။

21.2.7. Metadata ဂုဏ်သတ္တိများ (Metadata Properties)

Metadata tab တွင် layer နဲ့ပတ်သက်သည့် Metadata report ကို ဖန်တီးခြင်းနှင့် ပြင်ဆင်ခြင်းများ လုပ်ဆောင်နိုင်ပါသည်။ ပိုမိုသိရှိလိုသည်များအတွက် Metadata တွင် ကြည့်ရှုပါ။
