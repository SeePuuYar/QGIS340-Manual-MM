24.2. အချိန်နှင့်တစ်ပြေးညီ လမ်းကြောင်းဆွဲခြင်း (Live GPS tracking)

  • GPS toolbar

  • GPS အချက်အလက်ပြ Panel (GPS Information Panel)

    • နေရာနှင့်အပိုအချက်အလက်များ (Position and additional attributes)

    • အချက်ပြလှိုင်း (Signal)

  • Connect to a Bluetooth GPS for live tracking (Live လမ်းကြောင်းခြေရာခံရန်အတွက် bluetooth GPS သို့ချိတ်ဆက်ခြင်း)

  • QGIS ထဲတွင် GPS device များချိတ်ဆက်မှု ဥပမာများ (Examples of GPS devices connection in QGIS)

    • GPSMAP 60cs ကိုအသုံးပြုခြင်း (Using GPSMAP 60cs)

      • မိုက်ခရိုဆော့(ဖ်) ဝင်းဒိုး စနစ် (MS Windows)

      • Ubuntu/Mint GNU/Linux

    • BTGP-38KM datalogger ကိုအသုံးပြုခြင်း (Bluetooth ပစ္စည်းများအတွက်သာ)

    • BlueMax GPS-4044 datalogger ကိုအသုံးပြုခြင်း (Bluetooth နှင့် USB ၂ မျိုးလုံးအတွက် အလုပ်လုပ်သည်)

      • မိုက်ခရိုဆော့(ဖ်)ဝင်းဒိုး စနစ် (MS Windows)

      • Ubuntu/Mint GNU/Linux

QGIS တွင် GPS receiver တစ်ခုဖြင့် ကွင်းမြေပုံရေးဆွဲခြင်းလုပ်ဆောင်နိုင်ပါသည်။ ထိုကဲ့သို့သော အချိန်နှင့်တစ်ပြေးညီ လမ်းကြောင်းရေးဆွဲခြင်းကို GPS Toolbar ကိုအသုံးပြုပြီးလုပ်ဆောင်နိုင်ပါသည်။ QGIS နှင့် GPS receiver တို့မချိတ်ဆက်မီတွင် device configuration အချို့ကို လုပ်ဆောင်ရန် လိုအပ်နိုင်ပါသည်။

24.2.1. GPS toolbar

GPS Toolbar တွင် အချိန်နှင့်တပြေးညီလမ်းကြောင်းရေးဆွဲခြင်းကို ထိန်းချုပ်ရန် အဓိက tool များပါရှိပါသည်။ ၎င်းကို View ► Toolbars ► GPS မှ activate လုပ်နိုင်ပါသည်။ ၎င်းသည် project ၊ GPS ၊ နှင့် လက်ရှိ GPS track အခြေအနေကိုလိုက်ပြီး အဓိပ္ပါယ်ရှိသောအခါမှသာ action များကို လုပ်ဆောင်ပါသည်။ ကိရိယာကို QGIS မှ ရှာဖွေတွေ့ရှိသည်နှင့် အောက်ပါတို့ကို အပြန်အလှန်လုပ်ဆောင်နိုင်ပါလိမ့်မည်-

  • Connect to GPS (GPS သို့ချိတ်ဆက်ခြင်း)

  • Disconnect from GPS (GPS နှင့်အချိတ်အဆက်ဖြုတ်ခြင်း)

  • Recenter map on GPS location - အလိုအလျောက် အလယ်ဗဟိုပြန်လုပ်ခြင်း parameter ကိုထည့်သွင်းမစဉ်းစားပဲ လက်ရှိ GPS တည်နေရာတွင် မြေပုံကိုချက်ချင်း ဗဟိုပြုပါလိမ့်မည်။

  • Set destination layer for GPS digitized features - Feature digitize ပြုလုပ်သောအခါ QGIS သည် Default အနေဖြင့် active layer ကိုကြည့်ပါလိမ့်မည်၊ ဆိုလိုသ်ညမှာ GPS digitizing tool များသည် Layers panel ထဲတွင် select လုပ်ထားသော layer တွင်သာလုပ်ဆောင်ပြီး ဖန်တီးထားသော feature များကို ၎င်း layer ထဲတွင် သိမ်းဆည်းပါလိမ့်မည်။ ဤအရာသည် အချို့သောအကြောင်းအရာများတွင် အသုံးဝင်နိုင်ပါသည်၊ ထို့အပြင် ၎င်းသည် မလိုအပ်သော layer များထဲတွင် feature များမတော်တဆ သိမ်းဆည်းမိခြင်းမဖြစ်စေရန် အခြားသော layer များနှင့်လည်း ဂရုတစိုက်အပြန်အလှန်လုပ်ဆောင်ရန်အတွက်လည်း ရည်ရွယ်ပါသည်။ ဤ option သည် အချိန်နှင့်တပြေးညီခြေရာခံခြင်းအတွင်း data သိမ်းဆည်းရန်အတွက် layer ကို ရှင်းရှင်းလင်းလင်းညွှန်ပြစေနိုင်ပါသည်၊ သင်လိုအပ်ချက်ပေါ်မူတည်ပြီး ပြောင်းလဲပေးနိုင်ပါသည်။ Tool များသည် ရွေးချယ်ထားသော layer အမျိုးအစားနှင့်လည်း လိုက်လျောညီထွေဖြစ်စေပါလိမ့်သည်။

    Feature တစ်ခုကို ဖန်တီးသောအခါ GPS destination layer ကို အလိုအလျောက် တည်းဖြတ်ပြင်ဆင်နိုင်အောင် လုပ်ဆောင်ပေးပြီး အသုံးပြုသူကိုလည်း အကြောင်းကြားပေးပါသည်။

  • Feature ဖန်တီးခြင်းအတွက် action များ-

    • Add vertex to GPS track using GPS current location (GPS လက်ရှိတည်နေရာကိုအသုံးပြုပြီး GPS track တွင် vertex ထည့်သွင်းခြင်း)

    • Create a new point feature at current GPS location (လက်ရှိ GPS တည်နေရာတွင် point feature အသစ်တစ်ခုဖန်တီးခြင်း)

    • Create a new line feature using the current GPS track (လက်ရှိ GPS track ကိုအသုံးပြုပြီး line feature အသစ်တစ်ခုဖန်တီးခြင်း)

    • Create a new polygon feature using the current GPS track (လက်ရှိ GPS track ကိုအသုံးပြုပြီး polygon feature အသစ်တစ်ခုဖန်တီးခြင်း)

    • Reset track (Track ကို အစမှပြန်စခြင်း)

  • Show GPS information - GPS Information panel ကိုဖွင့်ပေးပါသည်

  • သင်ကြည့်လိုသော GPS အချက်အလက်များကို အလျင်အမြန်ကြည့်ရှုရန်အတွက် display box တစ်ခုတွင်-

    • Show location

    • Show altitude (Geoid)

    • Show altitude (WGS 84 ellipsoid)

    • Show ground speed

    • Show bearing

    • Show total track length

    • Show distance from start of track

  • Settings ခလုတ်ကိုနှိပ်လျှင် အသုံးများသော setting များပါဝင်သည့် drop-down menu တစ်ခုကို ပွင့်လာပါမည်၊ များသောအားဖြင့် ထို setting များကို session အလယ်ပိုင်းတွင် မွမ်းမံပြင်ဆင်ကြပါသည်-

    • Show location marker

    • Show bearing line

    • Rotate map to match GPS direction

    • မြေပုံ recentering ကိုထိန်းချုပ်ရန် ရွေးချယ်စရာများမှာ-

      • Always recenter map - GPS တည်နေရာသည် မြေပုံမြင်ကွင်းအလယ်မှ အကွာအဝေးတစ်ခု (မြေပုံမြင်ကွင်း extent ၏ အချိုးတစ်ခုအနေဖြင့်) ရွေ့သွားသောအခါ မြေပုံကို အမြဲတမ်း recenter (ဗဟိုပြန်လုပ်) ပြုလုပ်မည်ဖြစ်သည်။

      • Recenter map when leaving extent - extent ကိုကျော်လွန်သောအခါ မြေပုံကို recenter ပြုလုပ်ပါမည်။

      • Never recenter - ဘယ်သောအခါ မြေပုံကို recenter မပြုလုပ်ပါ။

    • Automatically add track vertices - GPS ကိရိယာမှ တည်နေရာအသစ်များ လက်ခံရရှိသည့်အခါတိုင်း track vertex များအလိုအလျောက်ထည့်သွင်းပါသည်။

    • Automatically save added features - GPS တည်နေရာများမှ ဖန်တီးထားသော feature များကို ၎င်းတို့၏ targer layer များသို့ ချက်ချင်းပို့ဆောင်ပါသည် (လုပ်နေကျ layer edit အဆင့်ကိုကျော်လိုက်ပြီး)

    • Time stamp destination - time fix သိမ်းဆည်းရန်အတွက် field ကို ချိန်ညှိရန်

    • Log to GeoPackage/SpatiaLite… - အမှန်ခြစ်ထားသောအခါ အသုံးပြုသူအား ရှိနေပြီးသား GeoPackage/SpatiaLite ဖိုင်တစ်ခုကိုရွေးချယ်ရန် သို့မဟုတ် ဖိုင်နာမည်အသစ်တစ်ခုထည့်ပေးရန် မေးမြန်းပါလိမ့်မည်။ gps_points နှင့် gps_tracks ဇယားတစ်ခုကို ကြိုတင်သတ်မှတ်ထားသောဖွဲ့စည်းပုံဖြင့် ဖိုင်ထဲတွင် ဖန်တီးပေးပါလိမ့်မည်။

      ဝင်ရောက်လာသော GPS message များအားလုံးကို GPS မှ အမြန်နှုန်း၊ လားရာ၊ အမြင့်နှင့် တိကျမှုအချက်အလက်များနှင့်အတူ gps_points layer တွင် မှတ်တမ်းတင်ပါလိမ့်မည်၊

      GPS သည် အချိတ်အဆက်ပြုတ်သွားသောအခါ (သို့မဟုတ် QGIS ကိုပိတ်လိုက်သောအခါ) record လုပ်ထားသော GPS track တစ်ခုလုံးကို gps_tracks ဇယားထဲသို့ ထည့်သွင်းပေးပါလိမ့်မည် (track အလျား၊ စချိန်နှင့် ဆုံးချိန် ကဲ့သို့ တွက်ချက်ထားသောအချက်အလက်အချို့နှင့်အတူ)။

    • Log NMEA sentences… - ကိရိယာမှ raw NMEA စာသားများအားလုံးကို text ဖိုင်တစ်ခုထဲသို့ မှတ်တမ်းသွင်းပေးပါသည်။

    • GPS settings… - GPS global options dialog ကိုအသုံးပြုစေနိုင်ပါသည်။

Tip

တည်နေရာအချက်အလက်အကြောင်း အချိန်နှင့်တပြေးညီပြသသည့်ဘား

GPS စက်တစ်လုံးကိုချိတ်ဆက်လိုက်ပြီး mouse cursor ကို map canvas ပေါ်တွင် ရွှေ့လိုက်သောအခါ mouse cursor မှ GPS တည်နေရာသို့ အကွာအဝေးနှင့် bearing (လားရာအညွှန်းထောင့်) ကို live (အချိန်နှင့်တပြေးညီ) message ဖြင့် ပြသပေးပါသည်။ Project အကွာဝေးနှင့် bearing အပြင်အဆင်များကို ဤမျက်နှာပြင်တွင် ပြသပေးပါသည်။ Touch screen အသုံးပြုနိုင်သော စက်များတွင် live status ကိုကြည့်ရန်အတွက် ထိပြီးဖိထားလိုက်ပါ။

24.2.2. GPS အချက်အလက်ပြ Panel (GPS Information Panel)

QGIS ထဲတွင် အချိန်နှင့်တပြေးညီခြေရာခံခြင်းတစ်ခုကို အပြည့်အဝစောင့်ကြည့်ရန်အတွက် GPS Information Panel ကိုဖွင့်ထားပေးရန်လိုအပ်ပါသည် (View ► Panels ► သို့မဟုတ် Ctrl+0 ကိုနှိပ်ပါ)။

QGIS နှင့် plug တပ်ထားသော GPS ကိရိယာအကြား ချိတ်ဆက်မှုစတင်ရန် GPS Information Panel ၏ ညာဘက်အပေါ်ထောင့်ရှိ Connect ကိုနှိပ်ပါ၊ သို့မဟုတ် အချိတ်အဆက်ဖြုတ်ရန် Disconnect ကိုနှိပ်ပါ။

Panel ၏ ဘယ်ဘက်အပေါ်တွင် အောက်ပါခလုတ်များရှိပါသည်-

  • Position - GPS တည်နေရာနှင့် sensor များအကြောင်း အသေးစိတ်တိုက်ရိုက်ပြသမှု

  • Signal - ဂြိုလ်တုချိတ်ဆက်မှုများ၏ အချက်ပြမှုပြင်းအား (signal strength)

  • Settings - Session တစ်ခုအတွင်း မွမ်းမံပြင်ဆင်ရန် လိုအပ်ကောင်းလိုအပ်နိုင်မည့် live tracking options drop-down menu

24.2.2.1. နေရာနှင့်အပိုအချက်အလက်များ (Position and additional attributes)

Position tab ထဲတွင် GPS သည် ဂြိုလ်တုမှ အချက်ပြမှုများကို လက်ခံရယူနေလျှင် တည်နေရာ၏ လတ္တီကျု၊ လောင်ဂျီကျု နှင့် အမြင့်များကို ထပ်ပေါင်းအချက်အလက်များနှင့်အတူ တွေ့မြင်ရပါမည်။

  • Latitude

  • Longitude

  • Altitude (Geoid) - ပျမ်းမျှပင်လယ်ရေမျက်နှာပြင် အထက် သို့မဟုတ် အောက်ရှိ Altitude/elevation

  • Altitude (WGS 84 ellipsoid) - WGS-84 Earth ellipsoid အထက် သို့မဟုတ် အောက်ရှိ Altitude/elevation

  • Geoidal separation - WGS-84 Earth ellipsoid နှင့် ပျမ်းမျှပင်လယ်ရေမျက်နှာပြင် (geoid) အကြားခြားနားချက်၊ - သည် ပျမ်းမျှပင်လယ်ရေမျက်နှာပြင်သည် ellipsoid အောက် ရောက်သည်ဟု ဆိုလိုပါသည်။

  • Time of fix

  • Speed - မြေပြင်အမြန်နှုန်း

  • Direction - True north မှ သွားလာသည့်လားရာသို့ နာရီလက်တံအတိုင်း ဒီဂရီဖြင့်တိုင်းတာသော Bearing

  • HDOP - Horizontal dilution of precision (ရေပြင်ညီအတိုင်း တိကျမှု လျော့ကျခြင်း)

  • VDOP - Vertical dilution of precision (ဒေါင်လိုက်အတိုင်း တိကျမှု လျော့ကျခြင်း)

  • PDOP - Dilution of precision (တိကျမှု လျော့ကျခြင်း)

  • H accuracy - မီတာဖြင့် ရေပြင်ညီတိကျမှု

  • V accuracy - မီတာဖြင့် ဒေါင်လိုက်တိကျမှု

  • 3D accuracy - မီတာဖြင့် 3D Root Mean Square (RMS)

  • Mode - GPS receiver configuration 2D/3D mode ၊ automatic သို့မဟုတ် manual ဖြစ်နိုင်ပါသည်

  • Dimensions - Position fix dimension ၊ 2D ၊ 3D သို့မဟုတ် No fix ဖြစ်နိုင်ပါသည်

  • Quality - တည်နေရာပြသမှု အရည်အသွေးအညွှန်း

  • Status - Position fix status ၊ Valid သို့မဟုတ် Invalid ဖြစ်နိုင်ပါသည်

  • Satellites - မူသေတည်နေရာရရှိရန်အတွက် အသုံးပြုသော ဂြိုလ်တုအရေအတွက်

  • Total track length - လက်ရှိ GPS track ၏ စုစုပေါင်းအကွာအဝေး

  • Distance from start of track - လက်ရှိ GPS track ရှိ ပထမဆုံး vertex မှ နောက်ဆုံး vertex သို့ တိုက်ရိုက်အကွာအဝေး

24.2.2.2. အချက်ပြလှိုင်း (Signal)

Signal tab ထဲတွင် အချက်ပြလှိုင်းများ ထုတ်လွှင့်နေသော ဂြိုလ်တုများ၏ အချက်ပြမှုပြင်းအား (signal strength) ကို မြင်တွေ့နိုင်ပါသည်။

24.2.3. Connect to a Bluetooth GPS for live tracking (Live လမ်းကြောင်းခြေရာခံရန်အတွက် bluetooth GPS သို့ချိတ်ဆက်ခြင်း)

ကွင်း data များကောက်ယူစုဆောင်းခြင်းအတွက် QGIS နှင့် bluetooth GPS ကိုချိတ်ဆက်နိုင်ပါသည်။ ထိုသို့လုပ်ဆောင်ရန် GPS Bluetooth device တစ်ခုနှင့် ကွန်ပျူတာတွင် Bluetooth receiver တစ်ခုလိုအပ်ပါသည်။

ပထမဆုံးအနေခြင့် GPS device ကို ကွန်ပျူတာက သိအောင်လုပ်ပြီး ကွန်ပျူတာနှင့် ချိတ်ဆက်ရပါမည်။ GPS ကိုဖွင့်ပြီး notification area ထဲရှိ Bluetooth icon ကိုသွား၍ device အသစ်တစ်ခုကိုရှာပါ။

အသုံးပြုနိုင်သောစာရင်းထဲတွင် မိမိ၏ GPS ပေါ်နေစေရန် Device ၏ ညာဘက်ခြမ်းတွင် ရွေးချယ်ခြင်း mask သည် device များအားလုံးရွေးချယ်ထားခြင်းကို သေချာအောင်လုပ်ဆောင်ပေးပါသည်။ နောက်တစ်ဆင့်တွင် serial connecion service ရရှိသင့်မည်ဖြစ်ပြီး ၎င်းကိုရွေးချယ်ကာ Configure (စိတ်ကြိုက်ပြင်ဆင်ခြင်း) ခလုတ်ကိုနှိပ်ပါ။

Bluetooth property များဖြင့်ရလာသော GPS ချိတ်ဆက်မှုကိုသတ်မှတ်ပေးထားသော COM port အရေအတွက်ကို မှတ်သားထားပါ။

GPS ကို ကွန်ပျူတာမှ သိရှိပြီးသည်နှင့် ချိတ်ဆက်ခြင်းလုပ်ဆောင်ပါ။ များသောအားဖြင့် ချိတ်ဆက်ရန်အသုံးပြုသော ဂဏန်းမှာ 0000 ဖြစ်ပါသည်။

GPS information panel ကိုဖွင့်ပြီး GPS options screen သို့ပြောင်းလဲပါ။ GPS ချိတ်ဆက်မှုအတွက် သတ်မှတ်ပေးထားသော COM port ကိုရွေးချယ်ပြီး Connect (ချိတ်ဆက်ပါ) ကို နှိပ်ပါ။ ခဏကြာပြီးနောက် မိမိ၏တည်နေရာကိုဖော်ပြနေသည့် cursor တည်နေရာတစ်ခု ပေါ်လာပါလိမ့်မည်။

QGIS သည် GPS data များကိုလက်မခံနိုင်လျှင် GPS ကို ပိတ်ပြီးပြန်ဖွင့်သင့်ပါသည်။ ၅ စက္ကန့်မှ ၁၀ စက္ကန့်လောက်ထိ စောင့်ပြီး ထပ်မံချိတ်ဆက်ကြည့်ပါ။ များသောအားဖြင့် ဒီနည်းလမ်းဖြင့် ပြဿနာကိုဖြေရှင်းနိုင်ပါသည်။ ချိတ်ဆက်မှုပြဿနာထပ်မံရှိနေဆဲဆိုလျှင် ၎င်း GPS ချိတ်ဆက်နေသော တခြား Bluetooth device တစ်ခု အနီးအနားတွင် ရှိမရှိ စစ်ဆေးပြီး ချိတ်ဆက်မှုဖြုတ်ပစ်ပါ။

24.2.4. QGIS ထဲတွင် GPS device များချိတ်ဆက်မှု ဥပမာများ (Examples of GPS devices connection in QGIS)

24.2.4.1. GPSMAP 60cs ကိုအသုံးပြုခြင်း (Using GPSMAP 60cs)

မိုက်ခရိုဆော့(ဖ်) ဝင်းဒိုး စနစ် (MS Windows)

၎င်းအလုပ်လုပ်ဆောင်ရန် အလွယ်ကူဆုံးနည်းလမ်းမှာ GPSGate <https://gpsgate.com/gpsgate-splitter> ဟုခေါ်သော middleware (freeware၊ not open) ကိုအသုံးပြုခြင်းဖြစ်ပါသည်။

Program ကိုဖွင့်ပြီး GPS device များကိုရှာပါ (USB နှင့် bluetooth ၂ မျိုးလုံးအတွက် အလုပ်လုပ်ပါသည်)၊ ထို့နောက် QGIS ထဲတွင်-

  1. Settings ► Options ► GPS ထဲတွင် plug တပ်ထားသော ကိရိယာကို ရှာဖွေပါ။ Autodetect mode ကိုအသုံးပြုနိုင်ပါသည်။

  2. GPS Information panel ထဲတွင် Connect ကိုနှိပ်ပါ။

Ubuntu/Mint GNU/Linux

Window ကွန်ပျူတာများအတွက်ကဲ့သို့ အလွယ်ကူဆုံးနည်းလမ်းမှာ အလယ်ရှိ server တစ်ခုကို အသုံးပြုရန်ဖြစ်ပြီး ယခု OS များအတွက်တော့ GPSD ဖြစ်ပါသည်။ ထို့ကြောင့် -

  1. Install the program

    sudo apt install gpsd

  2. ထို့နောက် garmin_gps kernel module ကိုခေါ်ယူထည့်သွင်းပါ။

    sudo modprobe garmin_gps

  3. ထို့နောက် unit ကိုချိတ်ဆက်ပါ။ Unit မှ အသုံးပြုနေသော တကယ့် device ကို dmesg ဖြင့်စစ်ဆေးပါ၊ ဥပမာ- /dev/ttyUSB0။

  4. ထို့နောက် gpsd ကို စတင်ဖွင့်နိုင်ပါပြီ။

    gpsd /dev/ttyUSB0

  5. နောက်ဆုံးအဆင့်အဖြစ် QGIS live tracking tool နှင့်ချိတ်ဆက်ပါ။

24.2.4.2. BTGP-38KM datalogger ကိုအသုံးပြုခြင်း (Bluetooth ပစ္စည်းများအတွက်သာ)

Linux အတွက် GPSD ကို အသုံးပြုခြင်း သို့မဟုတ် Window အတွက် GPSGate သည် အားစိုက်ထုတ်ရန်မလိုအပ်ပါ။

24.2.4.3. BlueMax GPS-4044 datalogger ကိုအသုံးပြုခြင်း (Bluetooth နှင့် USB ၂ မျိုးလုံးအတွက် အလုပ်လုပ်သည်)

မိုက်ခရိုဆော့(ဖ်)ဝင်းဒိုး စနစ် (MS Windows)

GPSGate သို့မဟုတ် ၎င်းမပါလျှင်ပင် live tracking သည် USB နှင့် Bluetooth ၂ မျိုးလုံးအတွက် အလုပ်လုပ်ပါသည်။ Autodetect mode ကို အသုံးပြုပါ သို့မဟုတ် မှန်ကန်သော port ကို ညွှန်ပေးပါ။

Ubuntu/Mint GNU/Linux

USB အတွက် (For USB)

Live tracking သည် QGIS live tracking tool ကို device နှင့် တိုက်ရိုက်ချိတ်ဆက်ခြင်းဖြင့် GPSD အသုံးပြုပြီးသော်လည်းကောင်း သို့မဟုတ် GPSD မသုံးပဲသော်လည်းကောင်း ၂ မျိုးလုံးနှင့် အလုပ်လုပ်ပါသည်။ (ဥပမာ- /dev/ttyACM3)
